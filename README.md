# Salary Predictor — Linear Regression from Scratch

Predicts salary based on years of experience using Linear Regression 
implemented from scratch in NumPy — no sklearn used for the model.

## What I built
- Linear regression with gradient descent coded in pure NumPy
- Trained on real salary dataset (30 samples)
- Streamlit UI with live salary prediction slider
- Learning curve and regression line visualisations

## Tech Stack
Python · NumPy · Pandas · Matplotlib · Streamlit

## How to run locally
pip install numpy pandas matplotlib streamlit
streamlit run app.py

## Live Demo
[https://sunaygourkhede-dev-salary-predictor-ml.streamlit.app/]

## Key concepts applied
- Gradient descent from scratch
- Feature normalisation (Z-score)
- MSE cost function
- Train/predict pipeline
